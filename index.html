Import into Nuvio

Nuvio is currently in beta, so these import steps may change.

Web login

  1. Log in to Nuvio.
  2. Select the correct profile.
  3. Open Collections.
  4. Click Import.
  5. Select the downloaded JSON file.
  6. Choose Add as new, Merge, or Overwrite.
  7. Click Add collections.

TV app

  1. Open Nuvio and choose a profile.
  2. Open Addons, then Collections.
  3. Select Import.
  4. Choose From File or From URL.
  5. For From File, load the JSON from Downloads, then confirm import.
  6. For From URL, enter the URL, fetch it, then confirm import.

For metadata enrichment, turn on the TMDB integration in Nuvio: Settings > Integrations > TMDB.
